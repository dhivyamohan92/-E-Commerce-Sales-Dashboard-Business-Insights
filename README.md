# Sales Dashboard and Data Analysis Using Microsoft Excel

## Description

This project analyzes retail sales data using **Microsoft Excel** to generate meaningful business insights. The dataset contains information such as customer details, product categories, store information, sales amounts, discounts, and payment methods.

The project includes data cleaning, descriptive statistical analysis, and the creation of an interactive dashboard using Excel features like Pivot Tables, Pivot Charts, Slicers, and formulas. The dashboard enables users to monitor sales performance and identify trends across different categories, regions, and customer segments.

---

# Getting Started

## Dependencies

Before using this project, ensure you have:

* Microsoft Excel 2019, Excel 2021, or Microsoft 365
* Windows 10/11 or macOS
* Basic knowledge of Excel formulas and Pivot Tables

---

# Installing

1. Download the Excel project file (.xlsx).
2. Open the file in Microsoft Excel.
3. Enable editing if prompted.
4. Refresh Pivot Tables if the data has been updated.

---

# Executing Program

### Step 1

Open the Excel workbook.

### Step 2

Review and clean the data by:

* Removing duplicates
* Handling missing values
* Checking data formats

### Step 3

Calculate descriptive statistics using Excel formulas:

* Mean
* Median
* Mode
* Minimum
* Maximum
* Count
* Variance (Sample and Population)
* Standard Deviation (Sample and Population)
* Skewness
* Kurtosis

### Step 4

Create Pivot Tables and Pivot Charts for:

* Total Sales
* Sales by Product Category
* Sales by Region
* Sales by Gender
* Sales by Payment Method
* Top Products
* Top Stores

### Step 5

Use Slicers and Filters to interact with the dashboard and analyze different business metrics.

# Help

### Common Issues

**Pivot Table is not updating**

* Right-click the Pivot Table and select **Refresh**.

**Charts are not reflecting new data**

* Refresh the Pivot Table or update the data source range.

**Formula errors**

* Verify that the cell references are correct and the data contains numeric values.
